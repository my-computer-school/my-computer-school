  • ubuntu 22.04 절전모드 해제 방법
    1. 터미널에 아래 명령을 입력해 보자.
    2. systemctl status sleep.target suspend.target hibernate.target hybrid-sleep.target
    3. Loaded: loaded 라고 적혀있으면 절전모드 동작 중이라는 뜻이다.
    4. sudo systemctl mask sleep.target suspend.target hibernate.target hybrid-sleep.target
    5. Loaded: loaded 에서 Loaded: masked 로 바뀐걸 볼 수 있다.
